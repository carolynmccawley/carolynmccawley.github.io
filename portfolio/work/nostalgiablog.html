Built with:
  • Node.
  • Express
  • EJS Templates
  • nedb Database

Website is hosted HERE, using Digital Ocean. The blog was created to illustrate the connection that people have through nostalgia; simply watching the same television shows can generate a relationships. The website takes is users comments and saves them in a database. The data is displayed using an EJS file that shows the data based on the network, show, and time.
