Travel Tracker: Firebase Web App

Web app created with Express and Node.js that uses Yelp and Google Places APIs

Travel Tracker is a web application that uses Firebase as a database to store user's login information and information about past and future trips. After logging in or creating an account with an email, username and password, a user is able to save data (including descriptions and notes) about past and upcoming trips. A user also has the ability to plan future trips; implemented with the Google Places and Yelp APIs, a user is able to research and save restaurants and attractions based on a given location.
