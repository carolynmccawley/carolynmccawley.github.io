Travel Tracker App

Built with:
  • React Native (JS)
  • Firebase
  • Yelp Fusion API
  • Google Places API

The Travel Tracker App is the mobile version of the Travel Tracker Web Application that Carolyn had previously built. The app stores the user's data, their saved restaurants, places, and notes, in a Firebase Database that can be accessed with their email and password. The user can search and save restaurants and places in a desired location. The app also keeps track of their notes; a person can create a note that saves the location, date, and any information they'd like to keep track of about a given trip.

While transitioning from web to mobile, Carolyn had to consider what changes needed to be implemented in lieu of the change in medium. Rather than having a navigation bar on top, she had the nav bar on the bottom of the screen. Additionally, she split the app up into multiple screens to adhere to the smaller screen size.
